How to Use

Method 1: Using the Interface

  1. Select the proxy protocol (HTTP, HTTPS, SOCKS4, SOCKS5)
  2. Enter the proxy server address
  3. Specify the port number
  4. Add username and password if required
  5. Click "Enable Proxy" to activate

Method 2: Using the Address Bar

Type in your browser's address bar:

https://setproxy/server:port

Examples:

  • https://setproxy/proxy.example.com:8080
  • https://setproxy/127.0.0.1:1080

With protocol and authentication:

https://setproxy/server:port?protocol=socks5&username=user&password=pass

To disable the proxy, type:

https://disableproxy

About ProxySwitch

ProxySwitch is a browser extension that allows you to quickly switch between different proxy servers. It supports multiple proxy protocols and provides both interface and URL-based control methods.

Features:

  • Support for HTTP, HTTPS, SOCKS4, and SOCKS5 proxies
  • Quick proxy switching via address bar
  • Optional authentication support
  • Settings persistence between browser sessions
